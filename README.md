# **Nathan Soares de Melo - Portfolio**

Welcome to my personal portfolio website. I am an **Engineering Student** at **ENSAE Paris**, specializing in **Data Science** and **Applied Statistics**. 

This portfolio highlights my academic background, technical skills, professional experiences, and personal interests including photography and music.

View the [live site here](https://nathansdm.eu).

## **About This Project**

This project is built with [Next.js](https://nextjs.org) and [Once UI](https://once-ui.com). It is based on the **[Magic Portfolio](https://github.com/once-ui-system/magic-portfolio)** template by Once UI.

## **Features**

- **Personalized Content**: tailored sections for About, Work, and Studies.
- **Gallery**: A showcase of my photography work.
- **Music**: A custom-built music player featuring my compositions and covers, integrating SoundCloud and Instagram.
- **Responsive Design**: Optimized for all devices using Once UI's design system.

## **Getting Started**

If you want to run this project locally:

**1. Clone the repository**
```bash
git clone https://github.com/nathsdm/my-portfolio.git
```

**2. Install dependencies**
```bash
npm install
```

**3. Run dev server**
```bash
npm run dev
```

## ** Tech Stack**

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Once UI](https://once-ui.com/) / CSS Modules
- **Deployment**: Hostinger
